Summary: Latitude vs X

The purpose of this project is to analyze how weather changes as you get closer to the equator. To do this analysis, we first pulled the OpenWeatherMap API to assemble a dataset on over 500 cities. After assembling the data, we used MatplotLib to plot various aspects of the weather vs. latitude.

Factors we looked at included: temperature, cloudiness, windspeed, and humidity. This site provides the source data and visualizations as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
